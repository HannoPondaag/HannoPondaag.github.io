  • achterhalen van je IP-address : die geef je door bv via WhatsApp zodat je tegenstander die kan invullen voor Net Memory.NET (IPV4-adres #2) of Vier-Op-Een-Rij (of Net Four In A Row, engelstalig).

    Je eigen Lokale ip-adres: je kunt dit vinden door 'ipconfig' in CMD or the command-prompt te typen om in een LAN te verbinden en te spelen.

    Spelen in een LAN (Local Area Network) of direct via een UTP-kabel werkt ietwat anders dan spelen op het WAN internet: Het ip-adres om te verbinden.., dan vul je het volgende in: je kan het verkrijgen door 'ipconfig' in CMD, the command-prompt te typen. Voor dat kan je eerst CMD intypen in het zoekveld op de Windows 10 taakbalk. In CMD zoek nauwkeurig naar: 'Wireless LAN adapter Wi-Fi' als je verbonden bent met internet via Wi-Fi.

    Het IPV4-adres van je tegenstander op het WAN: Dat kan je b.v. via Whatsapp of telefoon van hem of haar verkrijgen. Hij/zij kan het krijgen via https://www.ip-adres.nl/ : hij/zij zend het via WhatsApp of phone. Waarschijnlijk is er ook een MODEM instelling noodzakelijk (port-forwarding op port 1200), zie b.v. voor KPN gebruikers het document : 'opmerkingen voor KPN gebruikers'.

    Four In A Row Series:
    ---------------------------
    Over mijn Four In A Row Series: Resultaten voor KPN provider testcases in Nederland met twee verschillende Modems..

    Case #1; [Connect]-ing twee computers in een (Draadloos) thuisnetwerk situatie door het KPN Modem Sagemcom box B12 te gebruiken.
    Maak een aparte regel voor een Memory Game x spel of een Vier-op-een-rij versie in het Modem.
    Stel: jij bent Speler #1, je tegenstander Speler #2.
    Vervolgens zoekt Speler #1 het host adres op voor de internal host in te kunnen vullen, m.b.v. CMD.EXE (zoekvenster) & ipconfig.
    Speler #1 zoekt het external host adres op, dit vult Speler #2 later in het invoerveld van het bewuste programma.
    In het veld van het Modem bij Port forwarding IPV4 bij external host komt een sterretje: *.
    Dan drukt, als de data in het Modem is opgeslagen, Speler #2 [Listen]
    Speler #1 voert in het invoerveld het host adres van computer 2 v. Speler #2 zijn computer in, te vinden in het Modem. (Een IP adres uit de zogenaamde Privé-reeks 192.168.xxx.xxx, in mijn geval is dat 192.168.2.7)
    Speler #2 voert in het invoerveld van het programma op ,zeg, computer 2, het external host IP address in.
    Zodra beide spelers [Enter] hebben gedrukt, telt Speler #1 3..2..1.. en drukt [Connect], dan 1 of 2 sec. later drukt Speler #2 [Connect]
    Resultaat: Test geslaagd !

    Case #2; [Connect]-ing de computer's Modem KPN Experia 10A Box over ADSL & the internet met je tegenstander (op een Online.nl Provider Modem): binnen 'Een stad':
    Stel: jij bent Speler #1, je tegenstander Speler #2..
    Vul in het Modem bij Poort forwarding, nieuw item (of reeds bestaand item, bij herhalen van de test/procedure) als LAN Host in: het host adres van de interne host zoals verkregen zoals op de wijze van bij Case #1; in mijn geval 192.168.2.8 ..
    De regel voor mijn tegenstander in zijn Modem bleef gelijk, zoals ik die ooit tevoren op het Online.nl Modem had ingesteld voor hem (jaren geleden voor NetMemory.exe, geloof ik)..
    Vervolgens zoekt mijn tegenstander, Speler #2 zijn external IP host address op via..

  • achterhalen van je IP-address : ..en geeft dit mij, Player #1 door via Whatsapp of liever telefoon.
    Ikzelf, Speler #1, zoek ook mijn external host IP address op en doe hetzelfde..
    Speler #1 noteert het external host IP Address van Speler #2, die hij of zij via Whatsapp of telefoon krijgt van Speler #2 & Speler #2 noteert die van Speler #1, voor later te gebruiken dus..
    Beide Spelers drukken of klikken, met hun muisaanwijzer, vervolgens op [Listen] , terwijl Whatsapp- of telefoonlijn open is.
    Beide Spelers vullen de external IP addresses, die ze eerder b.v. via telefoon hebben verkregen van hun tegenstander, in in het invoerveld als eindpunt c.q. eindapparaat adres.
    Beide Speler drukken kort na elkaar [Enter] (de Whatsapp- of telefoonlijn blijft open voor de verificatie later)
    Speler #1 telt 3..2..1.. als hij [Connect] heeft gedrukt of geklikt met zijn of haar muisaanwijzer.
    Speler #2 drukt of klikt ook, kort erna bijv. 1 à 2 sec. later, op [Connect] ..
    Mijn broer, Speler #2 speelt en ik ook..
    Mijn broer wint het 1e spel, Net Four In A Row als test.
    Resultaat: test geslaagd! Verbinding bewerkstelligd.
