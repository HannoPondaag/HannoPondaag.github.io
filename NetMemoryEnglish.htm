    About my Memory Game(s) Series: Results for KPN provider testcases in the Netherlands with two different Modems..

    Case #1; [Connect]-ing two computers in a (Wireless) home network situation using KPN Modem Sagemcom box B12
    Make a rule for a Memory Game x game or a Net Four In A Row version in the Modem.
    Let's say: you are Player #1, your opponent Player #2.
    Vervolgens zoekt Speler #1 het host adres op voor de internal host in te kunnen vullen, m.b.v. CMD.EXE (zoekvenster) & ipconfig.
    Speler #1 zoekt het external host adres op, dit vult Speler #2 later in het invoerveld van het bewuste programma.
    In het veld van het Modem bij Port forwarding IPV4 bij external host komt een sterretje: *.
    Dan drukt, als de data in het Modem is opgeslagen, Speler #2 [Listen]
    Speler #1 voert in het invoerveld het host adres van computer 2 v. Speler #2 zijn computer in, te vinden in het Modem.
    Speler #2 voert in het invoerveld van het programma op ,zeg, computer 2, het external host IP address in.
    Zodra beide spelers [Enter] hebben gedrukt, telt Speler #1 3..2..1.. en drukt [Connect], dan 1 of 2 sec. later drukt Speler #2 [Connect]
    Resultaat: Test geslaagd !

    Opmerkingen voor KPN gebruikers. Notes for KPN users in dutch.


    Tips: -Type ipconfig at the command prompt to discover your ip4 address to use with one of the games.
            - all games: Make sure your font size is set to default 96 DPI when used in Windows Vista : this can be achieved; type font in control panel..
            - Memory: Don't forget to make first a Connect(ion) then Start !
            - Memory: Don't play too fast, keep e.g. a minimum time interval of one second between mouseclicks
            - Memory: You can really play it via internet and also by connecting a standard network cable (UTP) between 2 PC's or over a LAN with e.g. a friend.
            - Memory: If you play with two people use the same version of the program (Delphi or Qt or Java programmed). This is the only way the two instances or programs can communicate properly !
