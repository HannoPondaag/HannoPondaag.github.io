Net Lingo


  • Net Lingo : installatie pakket voor windows. Java 7 SE ,of hoger, moet eerst geinstalleerd zijn ! Lingo spelen tegen elkaar via het net!
  • Net Lingo java : voer dit bestand direct uit op bv uw apple pc/Mac OS X(na installatie van Java 7 SE, of hoger!). Lingo spelen voor 2 personen of teams.
    starten onder Linux: type in een terminalvenster in de map van het gedownloade bestand : java -jar NetLingo.jar
