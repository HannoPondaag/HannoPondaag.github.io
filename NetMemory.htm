  • Net Memory .NET : a .NET version for Windows, programmed with Visual Studio Community 2019/2017. Je eigen Lokale ip-adres: je kunt dit vinden door 'ipconfig' in CMD or the command-prompt te typen om in een LAN te verbinden en te spelen.

    Spelen in een LAN (Local Area Network) of direct via een UTP-kabel werkt ietwat anders dan spelen op het WAN internet: Het ip-adres om te verbinden.., dan vul je het volgende in: je kan het verkrijgen door 'ipconfig' in CMD, the command-prompt te typen. Voor dat kan je eerst CMD intypen in het zoekveld op de Windows 10 taakbalk. In CMD zoek nauwkeurig naar: 'Wireless LAN adapter Wi-Fi' als je verbonden bent met internet via Wi-Fi.

    Het IPV4-adres van je tegenstander op het WAN: Dat kan je b.v. via Whatsapp of telefoon van hem of haar verkrijgen. Hij/zij kan het krijgen via https://www.ip-adres.nl/ : hij/zij zend het via WhatsApp of phone. Waarschijnlijk is er ook een MODEM instelling noodzakelijk (port-forwarding op port 1200, misschien ook nog poort 1201; dit is een kwestie van een beetje experimenteren & testen als het niet mocht lukken met poort 1200 alléén), zie b.v. voor KPN gebruikers het document : 'opmerkingen voor KPN gebruikers'. Je hebt de keuze om te spelen met plaatjes van: dieren, vogels, bloemen, het alfabet, het grieks alfabet en getallen met verschillend aantal kaartjes alles geintegreerd in één programma.
    Klik met rechtermuisknop het Net Memory .NET icoon na installatie in windows' startmenu en selecteer meer..run as administrator om opgenomen spelletjes op te slaan en de beste spelers data!
    Opmerking: Gebruik de 'Zend' functie in het menu 'Spelen', mocht er een plaatje blijven liggen of in noodgevallen als het programma niet meer lijkt te reageren, terwijl de verbinding misschien nog wel bestaat!!!
    Als dit niet lukt vanaf de kant waar het probleem ontstaan is, probeer dan het overgebleven spel met de 'Zend' functie in het menu 'Spelen' te laten plaatsvinden vanaf de andere kant van de verbinding en probeer het spel te vervolgen. Als het goed is wordt de score ook meegezonden.. Veel geluk daarbij, in de praktijk blijkt het verzenden van actie nl. heel soms te falen (door het netwerk-verkeer). . (Ook een TCP/IP verbinding lijkt net als het UDP protocol niet altijd 100 % te voldoen op het WAN, na uitvoerig getest te zijn !). Mijn netwerk programma's gebruiken allemaal IPV4 type adresssen.
    Veel plezier !

    Over mijn Memory Game(s) Series: Resultaten voor KPN provider testcases in Nederland met twee verschillende Modems..

    Case #1; [Connect]-ing twee computers in een (Draadloos) thuisnetwerk situatie door het KPN Modem Sagemcom box B12 te gebruiken.

    Speler #1 zet UPNP IGD aan in het KPN Sagemcom Experia box B12 Modem/Router.
    Maak een aparte regel voor een Memory Game x spel of een Vier-op-een-rij versie in het Modem.
    Stel: jij bent Speler #1, je tegenstander Speler #2.
    Vervolgens zoekt Speler #1 het host adres op voor de internal host in te kunnen vullen, m.b.v. CMD.EXE (zoekvenster) & ipconfig.
    Speler #1 zoekt het external host adres op, dit vult Speler #2 later in het invoerveld van het bewuste programma.
    In het veld van het Modem bij Port forwarding IPV4 bij external host komt een sterretje: *.
    Dan drukt, als de data in het Modem is opgeslagen, Speler #2 [Listen]
    Speler #1 voert in het invoerveld het host adres van computer 2 v. Speler #2 zijn computer in, te vinden in het Modem. (Een IP adres uit de zogenaamde Privé-reeks 192.168.xxx.xxx, in mijn geval is dat 192.168.2.7)
    Speler #2 voert in het invoerveld van het programma op ,zeg, computer 2, het external host IP address in.
    Zodra beide spelers [Enter] hebben gedrukt, telt Speler #1 3..2..1.. en drukt [Connect], dan 1 of 2 sec. later drukt Speler #2 [Connect]
    Resultaat: Test geslaagd !

            - alle games: Zorg ervoor dat je font size indien gebruikt in Windows Vista 96 DPI is: dit kan in Windows Vista door: font in control panel te typen..
            - Memory: Let op. Eerst verbinden dan pas start !
            - Memory: Speel niet te snel, houd een tussenpoze van tenminste één seconde aan tussen muisklikken.
            - Memory: Je kan via internet spelen, door een standaard netwerk kabel (UTP) tussen twee PC's aan te sluiten óf via een LAN (lokaal netwerk).
            - Memory: Speel je met twee personen, dan moeten beiden dezelfde versie (.NET-, Delphi-, Qt- of Java- of C# geprogrammeerd) gebruiken anders is er geen communicatie mogelijk waarschijnlijk !
